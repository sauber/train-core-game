# Track

## Connect stations

A track connects two Stations directly. 

## Distance

It has a distance which is same as distance between Stations on Area.

## Degraded

Tracks degrade for every train pass. When fully degraded, Track is Broken, otherwise Operational.

## Building

Building a track as has cost: cost = distance * buildunitcost

## Repairing

Building a track as has cost: cost = distance * repairunitcost
