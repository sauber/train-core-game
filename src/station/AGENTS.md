# Station

Has a name.

Location is somewhere on Area.

Has 0 or more waiting passengers.

Has 0 or more tracks connecting to other stations.

Has a capacity for wating trains. The capacity is at least one, but grows
incrementally as more and more passengers have boarded or disembarked at the
station. There is an upper capacity limit. The Station Agent ensures platform
capacity increases.

Generates revenue when passengers exit. Revenue is added to Simulation Balance.
