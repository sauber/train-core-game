# Windows PowerShell Environment

This machine runs Windows 11 with PowerShell as the default shell.

Always use PowerShell-compatible commands:
- Expand-Archive / Compress-Archive for zip operations
- Get-ChildItem instead of ls
- Remove-Item instead of rm
- Copy-Item instead of cp
- Move-Item instead of mv
- Invoke-WebRequest instead of curl or wget
- Use PowerShell syntax for scripting (not bash)
